# 关于本文
Elrs（全称**ExpressLRS**），是一套软硬件开源的双向无线射频系统，旨在为OpenTX/EdgeTX 提供稳定且高性价比的开源射频方案。Elrs在航模/模型遥控场景应用较广，有丰富的硬件商品可供使用，并且其可靠性在广泛的实际应用中得到了验证。

笔者在本科期间有幸参与了Robocon赛事，随着近几年赛题对机器人的自动化要求不断提高，传统的以单向传输通道值的遥控方案越发局限。并且因实际比赛场地无线电环境复杂，WiFi、蓝牙，以及其他采用2.4Ghz通讯的无线模块（如nrf24）的稳定工作也面临更多不确定性。

基于以上实际应用需求，本项目围绕Elrs，阐述其相关工作过程及实际应用，旨在推广其在机器人竞赛、科研及相关产品的应用，为行业发展尽自己的微薄之力。

# 阅读说明

考虑到Elrs已在市场上充分落地，相关产品繁多，并且因不同读者对遥控方案的定制化程度要求的不同，本文将由浅到深，分章节阐述完成不同定制化程度所需要的相关知识，请阅读目录，按需进行阅读。

同时，笔者也会在分享一些与ELRS本身无关，但在机器人遥控应用中可能会用到的小技巧，读者可根据需要选择性阅读。

笔者能力有限，本文难免存在疏漏和错误，欢迎各位读者批评指正。
